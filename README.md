# CoffeeApp 

iOS application for ordering coffee and tea. Study project.

## Screens

- **Home** — promotions, categories, featured drinks
- **Detail** — drink detail card, size and quantity selection

## Tech Stack

- Swift
- SwiftUI
- MVVM
- async/await

## Getting Started

1. Clone the repository
2. Open `CoffeeApp.xcodeproj`
3. Select a simulator or device
4. Press `Cmd + R`
